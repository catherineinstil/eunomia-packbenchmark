How to Make Changes

All screens are plain HTML/CSS files — no build tools, no frameworks. Edit the file, push to GitHub, live site updates in ~1 minute.

Option A — GitHub Copilot CLI (recommended for non-developers)

Open a terminal in the project folder and describe changes in plain English. Copilot will edit the file, commit and deploy.

1

Open Terminal and navigate to the project

/Users/catherinemcconalogue/Projects/Eunomia
2

Start Copilot CLI and describe what you want

e.g. "Change the heading on screen-1-browser.html from Household Cleaning to Personal Care"

3

Copilot edits, commits and deploys automatically

Changes are live at the public URL within ~1 minute.

Option B — Edit directly in VS Code

1

Edit the relevant .html file in VS Code and save

2

Run this deploy command in Terminal from the project folder

git add -A && git commit -m "description of change" && git push origin iteration-2 && git checkout main && git merge iteration-2 && git push origin main && git checkout iteration-2

Key terminology

CAA — California Circular Action Alliance. The regulator / client. PRO user tier.
Touch — Packaging innovation agency. Uploads product records as data maintainer.
Eunomia — Environmental consultancy. Would operate the platform.
Eco-modulation fees — Variable fees paid by brands based on packaging efficiency.
Best in class — The lightest / most efficient packaging in a given category.

Important: All data is illustrative, for pitch purposes only. Real data will come from Touch's database once the NDA is in place.
