Overview

EcoPack is a packaging benchmarking concept built for Eunomia's pitch to CAA (California Circular Action Alliance). The work consists of HTML concept screens showing what the platform could look like — not a working prototype, but enough to make the concept feel real for the pitch.

All screens are live and accessible via a single URL. The source code lives in a GitHub repository.

How to Make Changes

All screens are plain HTML files — no build tools, no framework. You edit the file, save it, and push to GitHub. Changes go live automatically via GitHub Pages within ~1 minute.

Option A — Use GitHub Copilot CLI (recommended, no coding needed)

If you have GitHub Copilot CLI installed, navigate to the project folder and describe the change you want in plain English. Copilot will make the edit, commit and deploy it for you.

1

Get access to the repository

Ask Cat to add you as a collaborator on catherineinstil/eunomia-packbenchmark via GitHub Settings → Collaborators.

2

Clone the project

In your terminal: git clone https://github.com/catherineinstil/eunomia-packbenchmark.git

Then: cd eunomia-packbenchmark && git checkout iteration-2

3

Open Copilot and describe your change

Run gh copilot (or open the Copilot CLI) in the project folder. Describe what you want in plain English, e.g. "Change the heading on screen-1-browser from Household Cleaning to Personal Care".

4

Deploy

After Copilot makes the change, push to deploy:

git push origin iteration-2 && git checkout main && git merge iteration-2 && git push origin main && git checkout iteration-2

The live site updates within ~1 minute.

Option B — Edit the HTML directly

Open any .html file in a text editor (VS Code recommended). All content is clearly commented e.g. <!-- Floor Cleaners -->. Make your edit, save and push as above.

Always work on the iteration-2 branch, not main. Merge to main only when you're ready to publish. The deploy command above handles this automatically.

Design System

Colours

#16A34A — Primary green
#14532D — Dark green
#F0FDF4 — Light green bg
#DCFCE7 — Green chip fill
#FEF3C7 — Amber chip fill
#FEE2E2 — Red chip fill
#0F172A — Heading text
#F8F9FA — Page background

Typography

  • Font: Inter (Google Fonts) — loaded via CDN on all screens
  • Wordmark: font-size: 22px; font-weight: 700 — "Eco" in dark, "Pack" in green (#16A34A)
  • Page headings: font-size: 24px; font-weight: 700
  • Card names: font-size: 14px; font-weight: 700

Tier system

  • Green Best in class — meeting or beating the benchmark threshold
  • Amber Mid-range — some improvement needed
  • Red — Worst in class, significant improvement required

Product & Category Images

All images live in two folders within the project:

  • category-images/ — thumbnail images for category cards (e.g. hh-floor-cleaners.png)
  • product-images/ — individual product shots used on detail screens (e.g. Clinair Essig Reiniger)

To swap an image: replace the file in the relevant folder with the same filename, or update the src="" attribute in the HTML to point to the new file.

Data

All data on these screens is illustrative. Sample counts, tier distributions, benchmark values and product specs are representative placeholders based on Eunomia's PDF brief — not real CAA or Touch data. Do not present as factual without updating with real values.
